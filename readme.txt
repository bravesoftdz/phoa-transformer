PhoA Transformer v1.0 Readme, English
--------------------------------------------------------------------------------


DESCRIPTION
-----------
PhoA Transformer is a supplementary tool for handling PhoA photo albums
(PhoA is a free image cataloguing software available at http://www.dk-soft.org/ ).

The initial developer of the program is Mikhail Virovets <virovets@comtv.ru>

PhoA Transformer consists of the two different tools joined in the single user
interface:
  1) A converter of PhoA photo album data to XML file format and back;
  2) An XML report generator based on the XSLT technology.

The PhoA Transformer distribution includes some ready-to-use templates, in
particular for generating HTML pages with photos. You can edit the templates
directly using PhoA Transformer interface.

For more detailed description please refer to PhoA Transformer help file.


SYSTEM REQUIREMENTS
-------------------
It is recommended to have Microsoft Windows XP with Internet Explorer 6.0.

PhoA Transformer uses XSLT technology which requires the Microsoft XML Parser
(MSXML) version 3.0 or higher.
You may require to install MSXML (http://msdn.microsoft.com/xml) in order PhoA
Transformer to work properly in earlier Windows versions.

There are no special requirements for hardware though.
