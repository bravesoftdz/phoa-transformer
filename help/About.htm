Назначение

PhoA Transformer - это дополнительный инструмент для работы с фотоальбомом, созданным приложением PhoA (http://www.dk-soft.org).

Некоторые из применений PhoA Transformer описаны ниже.

Создание WEB-страниц с фотографиями Можно генерировать HTML файлы, включая в них любую информацию, содержащуюся в фотоальбоме.
Подготовка сводок и отчетов Можно представить в табличной форме данные из фотоальбома и метаданные EXIF с целью дальнейшего анализа.
Печать этикеток к фотографиям Можно распечатать связанную с каждой фотографией информацию для вклеивания в "реальный" фотоальбом.
Экспорт данных в другие программы Данные из фотоальбома и метаданные EXIF можно перенести, например, в Microsoft Excel.
Импорт данных из других программ Если описания фотографий, созданные в другой программе (например ASDSee), преобразовать в XML, то можно перенести их в фотоальбом PhoA.
Коррекция данных в фотоальбоме Если требуемая операция с данными не поддерживается приложением PhoA, можно скорректировать XML-файл и конвертировать его обратно в PhoA. (Пример: переименовать ключевое слово во всех изображениях.)
