Основное окно

Панель "Исходный"

Панель содержит окно XML-редактора и окно выбора группы.

Открыть
Вызывает диалог выбора исходного файла. Если выбранный файл имеет расширение xml, он будет открыт в окне редактирования "как есть". Если его расширение - phoa, то будет сперва запущен конвертор из PhoA в XML. Конвертор имеет окно настроек.
Сохранить
Если исходный файл имеет расширение xml, текст из XML-редактора сохраняется в этом файле. Если его расширение - phoa, будет запущен конвертор XML-текста из окна редактора в формат фотоальбома Phoa, и исходный файл будет перезаписан.
Сохранить как
Позволяет сохранить текст XML-редактора в файле с новым именем. Если выбранный файл имеет расширение phoa, будет запущен конвертор.
Окно выбора группы
Если в этом окне ничего не указано, преобразование применяется ко всему XML-документу, если выбрана одна из групп фотоальбома, преобразование применяется к этой группе (и, возможно, подгруппам). Конкретное поведение зависит от шаблона.

Панель "Шаблон"

Панель содержит окно редактора шаблона.

Открыть
Вызывает диалог выбора файла шаблона и открывает выбранный файл в окне редактора шаблона. По умолчанию, файлы шаблонов хранятся в папке
<папка содержащая PhoaTransform.exe>/Templates/xxx,
где xxx - обозначение языка шаблона (например, RUS - для русского).
Сохранить
Сохраняет текст из редактора шаблона.
Сохранить как
Сохраняет текст из редактора шаблона под новым именем.

Панель "Результат"

Панель содержит окно просмотра результата в виде текста и в виде WEB-страницы.

Преобразовать
Запускает преобразование исходного файла по выбранному шаблону. Результат преобразования помещается в окно просмотра результата.
Сохранить
Сохраняет текст из окна просмотра результата.
Сохранить как
Сохраняет текст из окна просмотра результата под новым именем.
Текст / HTML
Выбирает режим просмотра результата. Внимание! Режим HTML имеет смысл не для всех шаблонов.
